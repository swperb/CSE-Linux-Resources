CSE 1284: Introduction to Computer Programming

Introduction to Computer Programming is a course with a self-implied name that offers an introduction to the basics of programming using the Python language. It covers topics such as variables, loops, input/output, and basic data structures, with a focus on building basic problem solving skills through object-oriented programming.

About

As the creator of this website, my vision was to enhance the learning experience for CSE students here at MSU. The goal was and still is to fill in the gaps left from classes due to strict college schedules and offer supplemental material to enhance students' performance, not only in the classroom, but in personal programming experiences as well.
